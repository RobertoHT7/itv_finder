# ITV Finder — Frontend

Project scaffold created using Vite + React + TypeScript.
This frontend includes two pages (pestañas) implemented with react-router:

- / — Buscador de Estaciones (formulario de búsqueda + resultados)
- /carga — Carga del almacén de datos (selección de fuentes y acciones)

Quick start (Windows PowerShell):

```powershell
cd frontend
npm install
npm run dev
```

The dev server will start at http://localhost:5173 by default.

Navigation:

Open http://localhost:5173 to view the app. Use the top links to switch between the "Buscador" and "Carga de datos" pages.
