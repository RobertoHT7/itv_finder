# itv_finder

Buscador de estaciones ITV – Proyecto prácticas IEI

## Estructura del repositorio

- backend/ — API en TypeScript (Express + Supabase)
- frontend/ — Interfaz de usuario con React + Vite + TypeScript (inicializada)

## Cómo ejecutar (Windows PowerShell)

1. Backend

```powershell
cd backend
npm install
npm run dev
```

2. Frontend

```powershell
cd frontend
npm install
npm run dev
```

Visita http://localhost:5173 para ver el frontend en desarrollo.
